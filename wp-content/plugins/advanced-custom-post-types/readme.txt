Customise WordPress with powerful, professional and intuitive post types.

== Description ==

Advanced Custom Post Types is the perfect solution for any WordPress website which needs more content types like other Content Management Systems. Advanced Custom Fields v5+ (Pro) is required.

== Filters ==

**acpt/settings/show_admin** - Determines whether or not the admin menu is shown.

**acpt/settings/admin_fields** - Determines which field groups may appear in the post type editor.

**acpt/settings/capability** - Determines capability is needed to manage custom post types.

**acpt/post_types** - Alter the post types created

== Actions ==

**acpt/init** - Fires before acpt has started loading.
